不同风格的图像迁移#

2022.09 - 2022.12, Python / Pytorch / GANs

  • 通过 Python 和 PyTorch 针对图像风格迁移任务搭建了基于 GAN 的神经网络，使图像渲染时间节省了 50%，同时迁移结果在视觉一致性和保真度方面显著提升
  • 根据模型结果，以及其他基于 GAN 网络的模型进行对比，评估风格迁移技术，总结和比较四种不同算法模型在不同风格领域中的适用性和优势等，最终撰写论文综述

社交媒体网络应用开发#

2022.02 - 2022.06, Django / React / MySQL / Nginx / Oauth

  • 开发了一款基于 Django 和 React 的社交媒体网络应用，为超过 10,000 名用户提供强大的用户管理、动态帖子创建和互动式帖子评论等功能，将用户参与度提高了 35%
  • 基于 Django MVT 架构设计并实施了可扩展的后端，使用 MySQL 数据库配合存储过程以实现高效查询，使用 Nginx 进行负载均衡和代理以分发流量，并使用 Oauth 进行用户认证和日志记录保证用户数据安全
  • 搭建基于 React 的前端，使用 Axios.js 来进行 AJAX 请求，将页面加载时间缩短了25%，为用户提供了良好的用户界面和体验，同时与后端进行数据交互

电影盈利影响因素的数据可视化分析#

2022.01 - 2022.03, Pandas / TensorFlow / TextCNN

  • 使用Pandas对包含 2,600 万部电影的数据集进行分类和数据提取，从而简化了数据集的分析工作，将数据处理效率提高了 40%
  • 利用Python和TensorFlow构建和训练TextCNN网络，基于模型构建了一个电影推荐系统，捕捉用户特征，提供预测电影影响因素的模型支持
  • 实施Matplotlib和Plotly来制作数据可视化，为电影公司的受众定位、营销活动以及收入优化提供数据支持
